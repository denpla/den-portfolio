# den-portfolio
This is a project for Intec Brussels IT school. It's a portfolio site built with VUE, Vuetify, and VUEX.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
